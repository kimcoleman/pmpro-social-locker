== Description ==

Integrate Paid Memberships Pro with the Social Locker plugin from OnePress (http://wordpress.org/support/plugin/social-locker). The goal is to give a user a free membership if they interact with Social Locker.

== Installation ==

1. Upload the `pmpro-social-locker` directory to the `/wp-content/plugins/` directory of your site.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Define the PMPROSL_FREE_LEVEL_ID and PMPROSL_MEMBERSHIP_PERIOD_DAYS constants in a customizations plugin.
